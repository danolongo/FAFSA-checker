# FAFSA Automizer

FAFSA Automizer is a python script that checks the availability of the FAFSA form.

chromedriver is nessesary for this program
instructions for installation below, make sure to include chromedriver in the same folder as the project when usiing it
https://chromedriver.chromium.org/


## Usage

number of attempts adjustable in line 33
celebration video is modifiable in line 48, just input the url to whatever video you want to celebrate with
after proper adjustments, just run code and the chromedriver will automatically start to show the work being done
